Note 1: All syntax should be inside a double curly braces

Note 2: Conditions, range should have an end

Variable Assignment

Syntax: $v1 := 5

{{ $v1 := 5 }} {{ $v2 := 7 }}

Use call to invoke function in HTML templating from GoLang returned json.

Syntax: call .Multiply $v1 $v2

The product of {{ $v1 }} and {{ $v2 }} is {{call .Multiply $v1 $v2}}.

Conditional Statement.

If the index is 1 satisfy, else does not.

Syntax for if index = 1: if eq $index 1

    {{range $index, $element := .Fruits}} {{if eq $index 1}}
  • {{ $index }}: {{ $element }}: This satisfies the condition
    • {{ else }}
  • {{ $index }}: {{ $element }}: This does not satisfy the condition
    • {{ end }} {{end}}
